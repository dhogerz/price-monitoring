# List API

| API  | HTTP | Purpose |
| ------------- | ------------ | ------------- |

### Setup

```
$ mvn clean install
$ mvn spring-boot:run
```
Run on http://localhost:8000<br>

### Environment

```
JAVA 1.8
SpringBoot
PostgreSQL 
```

### Dependencies 

```
SpringBoot

```
